# riparian-avian-and-insect-monitoring

This repository contains the reproducible methods and results presented in the research article:

> Jacuzzi, G., Pekich, S., Olden, J.D. Aquatic insect subsidies mediate current and future urbanization effects on insectivorous birds across spatial scales

Results can be reproduced in sequence with the manuscript starting from script `src/4_msom.R`.
